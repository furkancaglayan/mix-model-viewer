◆ texture_filtering

enum class mix::texture::texture_filtering : int
strong
Enumerator
nearest 

GL_NEAREST is the default texture filtering method of OpenGL. When set to GL_NEAREST, OpenGL selects the texel that center is closest. Also known as point filtering.

linear 

GL_LINEAR takes an interpolated value from the texture coordinate's neighboring texels, approximating a color between the texels. The smaller the distance from the texture coordinate to a texel's center, the more that texel's color contributes to the sampled color. Also known as bi-linear filtering.
